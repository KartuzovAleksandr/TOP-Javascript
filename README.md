6. Встроенные объекты: даты, строки, регулярные выражения, коллекции
7. Обработка ошибок. BOM 
8. DOM
9. DOM. Работа с формами и событиями
10. JQuery
11. JQ UI. JSON. Работа c файлами. AJAX. Canvas
